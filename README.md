# Advanced Short URLs

[![License](https://img.shields.io/badge/license-CC0%201.0%20Universal-blue.svg)](https://creativecommons.org/publicdomain/zero/1.0/)
[![GitHub Issues](https://img.shields.io/github/issues/stopbot-net/smarturls.svg)](https://github.com/stopbot-net/smarturls/issues)

Advanced Short URLs by Moonito is a feature-packed URL shortening service that offers a wide range of capabilities for optimizing user experience, tracking visitors, and customizing link behavior. Powered by Moonito, this service ensures high-performance traffic management for web and mobile apps, while providing insightful analytics and protection from potential threats.

## Features

- **Create Custom URLs**: Easily generate advanced URLs with personalized rules and actions.
- **Flexible Redirection**: Send users to different destinations based on specific rules you set.
- **Multiple Redirection Methods**: Choose from direct links, iframes, or content loading without refreshing the page.
- **Simple Rule Setup**: Set up and adjust redirection rules with minimal effort.
- **Manage Everything in One Place**: Control all your URLs and settings through the Moonito panel.

To explore a detailed list of all the available features and learn more about how they can benefit you, please visit our full feature overview [here](https://go.moonito.net/evTJlA?utm_source=our-github).

## Requirements

- **Web Servers**: Apache or Nginx
- **PHP**: Version >= 8.1 with CURL extension enabled
- **Optimal Performance**:
  - Apache >= 2.4.9 or Nginx >= 1.26.2
  - PHP version >= 8.1
